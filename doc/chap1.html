1 The vngrape Package

This chapter describes the GAP package vngrape. It builds on the package GRAPE.

1.1 Functions

The following functions are available:

1.1-1 VNI2N
‣ VNI2N( graph, index_or_list_of_indices )( function )

converts all indices in x to vertex names

1.1-2 VNN2I
‣ VNN2I( graph, name_or_list_of_names )( function )

converts all names in x to its indices
